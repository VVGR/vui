vui.stick

Class: stick

stick

Author: putaoshu@126.com.
View source: stick.js.

Arguments:

vui.stick (options)

  1. options - (Object) 组件配置
  2. options.obj - (Object) null 粘到那个元素上
  3. options.stickTo - (Object) null 被粘的对象
  4. options.resize - (Boolean) true 窗口有变更自动更新
  5. options.autoShow - (Boolean) false 主体元素有被遮挡时,把主体转移到上面
  6. options.css - (Object)
  7. options.css.type - (Number) 0 输入偏移量,1为默认输入
  8. options.css.position - (String) 'absolute' position
  9. options.css.top - (Number) 0 向上的偏移量
  10. options.css.left - (Number) 0 向左的偏移量
  11. options.css.zIndex - (Number) 2 zIndex

Example: 

vui.stick({
	stick:$('#more'),
	stickTo:$('#moreOperateLayer'),
	autoShow:true,
	css:{
		top:3,
		left:0,
		zIndex:20
	}
});
